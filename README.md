# 华中地区数学建模竞赛记录
# Records of the Central China Mathematical Modeling Competition

A repository dedicated to recording and organizing participation records, solutions, and resources for the Central China Mathematical Modeling Competition.

---

## 📁 Repository Structure

```
├── competitions/          # Competition records organized by year
│   └── YYYY/              # Year folder (e.g., 2024/)
│       ├── README.md      # Summary of that year's competition
│       ├── problem-A/     # Problem A materials
│       └── problem-B/     # Problem B materials
├── resources/             # Reference materials and learning resources
│   ├── books/             # Recommended books and papers
│   └── tools/             # Useful tools and software guides
├── templates/             # Templates for recording competition entries
│   └── competition-record-template.md
└── README.md              # This file
```

---

## 🏆 Competition Overview

The **Central China Mathematical Modeling Competition** (华中地区数学建模联赛) is a regional mathematical modeling competition for college students in the Central China region. Teams of students apply mathematics to solve real-world problems over a defined contest period.

## 🚀 How to Contribute

1. **Add a new year's record:**
   - Create a folder under `competitions/YYYY/` (replace `YYYY` with the actual year).
   - Copy `templates/competition-record-template.md` into the new folder as `README.md`.
   - Fill in the details: problem description, approach, results, and reflections.

2. **Add resources:**
   - Place reference books, papers, or tool guides under the appropriate subfolder in `resources/`.

3. **Update the Records table** in this README with a summary of your entry.
