# Competition Records

This folder organizes competition participation records by year.

## Structure

Each year's records are stored in a subfolder named by the year (e.g., `2024/`).

```
competitions/
└── YYYY/
    ├── README.md        # Competition summary (copied from template)
    ├── problem-A/       # Materials for Problem A
    │   ├── report.pdf   # Final report
    │   └── code/        # Source code
    └── problem-B/       # Materials for Problem B
        ├── report.pdf
        └── code/
```

## Adding a New Record

1. Create a new folder: `competitions/YYYY/`
2. Copy `../templates/competition-record-template.md` as `README.md` inside the new folder.
3. Fill in all the fields.
4. Place your report and code in the appropriate subfolders.
5. Update the Records table in the root `README.md`.
