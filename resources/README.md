# Resources

This folder contains reference materials and guides useful for mathematical modeling competitions.

## Books

Place recommended textbooks, papers, or e-books in the `books/` subfolder.

**Suggested reading:**
- *Mathematical Modeling* — Mark M. Meerschaert
- *A First Course in Mathematical Modeling* — Frank R. Giordano et al.
- *Operations Research* — Frederick S. Hillier & Gerald J. Lieberman

## Tools

Place software guides, scripts, or tool documentation in the `tools/` subfolder.

**Commonly used tools:**
- **MATLAB** — Numerical computing and simulation
- **Python** (NumPy, SciPy, Matplotlib) — Data analysis and visualization
- **R** — Statistical modeling
- **LINGO / CPLEX** — Optimization problems
- **LaTeX** — Report writing
